Privacy Statement

Bail Bloc provides software that transforms computing power into cryptocurrency, which is then transformed into bail funds via our partner the Bronx Freedom Fund. 100% of the resources generated using the software are used to provide bail to defendants standing trial.

INFORMATION WE HAVE

We do not collect information of any participants. The Bail Bloc software does not collect or transmit any identifying information of the user's computer other than the IP address, which is used on a temporary basis by the mining pool (XMRpool.net) to prevent abuse of the network.

INFORMATION WE SHARE

We will not share any of your information with anyone unless required to by law.

OUR SECURITY

We use HTTPS, an encrypted protocol that establishes communication between the user's computer and our web server. Our code is open source so that it can be vetted by our community. The Bail Bloc software mines Monero, which is an anonymous and decentralized cryptocurrency.
